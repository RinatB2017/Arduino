I2C Bus

Accroding to different boards, user can use instances below directly, because these have been defined in Segnix.

On Iteaduino Plus and ITEAD CORE EVB
  • Wire - A reference to Wire1
  • Wire1 - I2C1(SDA:PB19, SCK:PB18)
  • Wire2 - I2C2(SDA:PB21, SCK:PB20)
On Raspberry Rv2 and Raspberry Pi Model B+
  • Wire - A reference to Wire1
  • Wire1 - I2C1(SDA:GPIO2, SCK:GPIO3)
On BeagleBoneBlack
  • Wire - A reference to Wire1
  • Wire1 - I2C1(SDA:GPIO0_4, SCK:GPIO0_5)
  • Wire2 - I2C2(SDA:GPIO0_12, SCK:GPIO0_13)
