uint32_t analogRead ( uint32_t  pin)

Get adc input via pin.

Parameters
pin- the adc input channel.
Returns
the adc result if possible.
Note
The result is an integer value. Its range depends on the ADC's precision. for example :[0 to 4095] — 12-bit ADC [0 to 1023] — 10-bit ADC

duty- 0-255.
Returns
